# 📚 gradegear
大学の授業における成績の確認と目標設定に役立つWebアプリ

- 学習の記録  
- 授業別の累計時間グラフ  
- 目標成績に向けた必要な残り点数  
- Web版はスマホのホーム画面に追加すれば PWA として使える  

Python学習・アプリ開発・就活ポートフォリオに最適。

---

# ✨ Features（機能）

## ✔ 記録管理
- 日付  
- 科目（例：線形代数、物理学基礎など）  
- 内容  
- 活動による評価/点数  

---

## ✔ 今日の提案（レコメンド）
目標成績のための残り点数の計算
**「すでに取った点数と全部取った場合の点数」** を検出して  
残り点数を提案する機能。
